HTML — это стандартный язык разметки для создания веб-страниц.
CSS — это язык, используемый для описания стиля документа, написанного на HTML или XML.
JavaScript — это язык программирования, соответствующий спецификации ECMAScript.
SCSS — это препроцессорный язык сценариев, который интерпретируется или компилируется в CSS.
Flexbox — это модель компоновки, которая позволяет элементам выравниваться и распределять пространство внутри контейнера.
Git — это распределенная система управления версиями для отслеживания изменений в исходном коде во время разработки программного обеспечения.
Bash — это Unix-оболочка и командный язык, написанный Брайаном Фоксом для проекта GNU.
TypeScript — это язык программирования, разработанный и поддерживаемый Microsoft.
React — это библиотека JavaScript для создания пользовательских интерфейсов.
Redux — это библиотека JavaScript с открытым исходным кодом для управления состоянием приложения.
Grid Layout — это двухмерная система компоновки для веба.
DOM — это программный интерфейс для веб-документов.
Webpack — это модульный сборщик JavaScript с открытым исходным кодом.
